Controls (Exact Java Port)

Menu Navigation:

  • N - New World (Random)
  • S - New World (Custom Seed)
  • L - Load Game
  • R - Replay Game
  • Q - Quit

Seed Input:

  • 0-9 - Type seed numbers
  • S or Enter - Start game with seed

In Game:

  • W A S D - Move avatar
  • : then Q - Save and quit to menu
  • Mouse hover - Show tile information

Note: Click on the game canvas area first to ensure keyboard input works properly.
